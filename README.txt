TMS320F28377D x uC-OS3 for TI CCS
=============================

This is a TI CCS project for TMS320F28377D with uC-OS3. It is based on TI's example project for uC-OS3.

The project is configured to use the F28377D CPU1. The CPU2 is not used.

The project is configured to use the F28377D peripheral drivers provided by TI.
